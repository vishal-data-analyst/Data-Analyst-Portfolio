
# 📊 Sales Data Analysis

## 📌 Objective
The objective of this project is to analyze sales data to identify trends, evaluate product performance, measure profitability, and generate business insights that can help improve decision-making.

---

## 🧾 Dataset Information
- Dataset Type: Sales / Superstore Dataset
- Records: Orders data including sales, profit, category, region, and customer details
- File Format: CSV / Excel

---

## 🛠️ Tools Used
- Microsoft Excel
- SQL
- Power BI

---

## 🧹 Data Cleaning Steps
- Removed duplicate records
- Checked for missing values
- Standardized column names
- Corrected data types
- Verified sales and profit values
- Cleaned category and region names

---

## 📊 Key Business Questions
This project answers the following questions:

1. What is the total sales and total profit?
2. Which products generate the highest sales?
3. Which regions perform the best?
4. What are the monthly sales trends?
5. Which categories are most profitable?
6. Which products or segments are causing losses?

---

## 🧠 Analysis Performed
- Sales trend analysis
- Profitability analysis
- Category-wise performance
- Region-wise performance
- Top products analysis
- Loss-making products identification

---

## 📈 Key Insights
- Identified top-performing products based on sales
- Found the most profitable categories
- Analyzed regional contribution to revenue
- Detected products and regions with low profitability
- Observed monthly patterns in sales performance

---

## 💡 Business Recommendations
- Focus more on high-performing products and categories
- Improve strategy in low-profit regions
- Review discounting strategy for loss-making items
- Increase inventory for best-selling products
- Monitor monthly trends for better planning
